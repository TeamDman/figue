runnable examples. Read it front to back the first time, then jump to whatever
you need.

2. [Fork Differences](@/guide/fork-differences.md) — intentional Teamy fork behavior that differs from upstream

11. [Help & Completions](@/guide/help-and-completions.md) — `--help`, `--html-help`, `--version`, shells
12. [JSON Schema Export](@/guide/json-schema.md) — editor autocomplete for config files
